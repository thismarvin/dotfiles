# dotfiles

a repository for all my dotfiles.

## Installation

I use [stow](https://www.gnu.org/software/stow/) to manage my dotfiles.

```bash
git clone https://github.com/thismarvin/dotfiles.git
cd dotfiles
stow -R -t $HOME nvim
```
